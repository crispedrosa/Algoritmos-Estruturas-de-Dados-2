# Bubble Sort (Aula 1)

O <i>Bubble Sort</i> é um dos algoritmos de ordenação mais simples, sendo executado percorrendo a lista várias vezes, comparando pares de elementos adjacentes. Caso o par estiver na ordem errada, ele é trocado, esse processo se repete até que nenhuma troca seja feita em uma passagem completa, indicando que a lista está ordenada. 

### Complexidade de Algoritmo

- **_Pior Caso (O(n^2 )):_** Ocorre quando a lista está em ordem decrescente.

- **_Melhor Caso (O(n)):_** Ocorre quando a lista já está ordenada.
